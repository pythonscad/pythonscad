# PythonSCAD APT Repository

This branch contains the APT repository for PythonSCAD Debian packages.

Visit the repository at: https://pythonscad.github.io/pythonscad

## Structure

- `dists/stable/` - Repository metadata
- `pool/main/p/pythonscad/` - Package pool
- `pythonscad-archive-keyring.gpg` - Public GPG key
- `index.html` - Repository homepage
